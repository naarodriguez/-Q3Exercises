HTML

HyperText Markup Language

HTML or hypertext markup languange is the basic need in making a website or a webpage. It is accompanied by other apps in order to make a page or site. An HTML file can be viewed through browsers after it is done.

CSS

Cascading Style Sheets

The css is used to decorate or style and html document. CSS has three different types, inline, internal, and external. Each type has a specific location for you to put the css in.

Bootstrap Version 4

Bootstrap is a way to improve or adjust the file depending on your device. This can be used from smartphones to desktops. Bootstrap may be downloaded online or offline.
